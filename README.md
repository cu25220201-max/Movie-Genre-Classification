# Movie Genre Classification

## Objective

The goal of this project is to classify movies into different genres based on their plot descriptions using Machine Learning.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorizer
* Logistic Regression

## Dataset

The dataset contains movie plot descriptions and their corresponding genres.

## Features

* Data Cleaning
* Text Vectorization
* Model Training
* Genre Prediction

## Model Used

Logistic Regression

## Output

The model predicts the genre of a movie based on the entered plot description.
