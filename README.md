#  Movie Genre Classification using Machine Learning

## Objective

The goal of this project is to classify movies into different genres based on their plot descriptions using Machine Learning.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorizer
* Logistic Regression

## Dataset

The dataset contains movie plot descriptions and their corresponding genres.

## Supported Genres

- Action
- Romance
- Horror
- Thriller
- Comedy


## Features
- Data Cleaning and Preprocessing
- TF-IDF Text Vectorization
- Machine Learning Classification
- Accuracy Evaluation
- Classification Report
- Confusion Matrix
- Genre Distribution Visualization
- Confidence Score Prediction
- Model Saving and Loading
- Prediction History Storage

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. TF-IDF Feature Extraction
5. Train-Test Split
6. Logistic Regression Model Training
7. Model Evaluation
8. Real-Time Genre Prediction
9. Model Saving using Joblib


## Model Used

### Logistic Regression

Logistic Regression was used as the classification algorithm because it performs well on text classification tasks and provides probability-based predictions.

## Evaluation Metrics

- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix
- 
## Output

The model predicts the genre of a movie based on the entered plot description.
